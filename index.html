1. GET главной страницы в HTTP 1.0

Метод GET запрашивает главную страницу. Протокол HTTP 1.0 после ответа сразу закрывает соединение (Connection: close).

GET / HTTP/1.0

2. GET внутренней страницы в HTTP 1.1

В протоколе HTTP 1.1 обязателен заголовок Host. Запрошена страница index.php через подпапку домена.

GET /u82689.kubsu-dev.ru/index.php HTTP/1.1
Host: u82689.kubsu-dev.ru

3. Размер файла file.tar.gz (HEAD)

Метод HEAD запрашивает только заголовки. Размер файла указан в заголовке Content-Length: 11335 байт.

HEAD /u82689.kubsu-dev.ru/file.tar.gz HTTP/1.1
Host: u82689.kubsu-dev.ru

4. Медиатип ресурса /image.png

Метод HEAD возвращает заголовок Content-Type: image/png — это медиатип PNG-изображения.

HEAD /u82689.kubsu-dev.ru/image.png HTTP/1.1
Host: u82689.kubsu-dev.ru

5. Отправка комментария методом POST

Метод POST отправляет данные на сервер. Заголовок Content-Type: application/x-www-form-urlencoded указывает формат данных. Content-Length: 14 — длина строки comment=Hello. Сервер вернул массив с комментарием — запрос обработан успешно.

POST /u82689.kubsu-dev.ru/index.php HTTP/1.1
Host: u82689.kubsu-dev.ru
Content-Type: application/x-www-form-urlencoded
Content-Length: 14

comment=Hello

6. Первые 100 байт файла

Заголовок Range: bytes=0-99 запрашивает часть файла. Сервер отвечает статусом 206 Partial Content и заголовком Content-Range.

GET /u82689.kubsu-dev.ru/file.tar.gz HTTP/1.1
Host: u82689.kubsu-dev.ru
Range: bytes=0-99

7. Кодировка ресурса index.php

Метод HEAD возвращает заголовок Content-Type с указанием кодировки. В ответе: charset=UTF-8.

HEAD /u82689.kubsu-dev.ru/index.php HTTP/1.1
Host: u82689.kubsu-dev.ru
